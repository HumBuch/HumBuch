Filter

Mit Hilfe von dem Feld „Filter“ kann speziell nach Schülernamen gesucht werden, falls eine unüberschaubare Schüleranzahl in der Tabelle gelistet ist.

Entliehene Materialien Anzeigen

Wenn dieser Button gedrückt wird, werden Bücher angezeigt, die der markierte Schüler aktuell ausgeliehen hat.

Importieren

Durch klick auf diesen Buttons öffnet sich der Windows Explorer, mit dem eine Schülerdatenbank zum Import ausgewählt werden kann.
