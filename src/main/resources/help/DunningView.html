Wobei benötigen Sie Hilfe?

  • Wozu dient die Tabelle?
  • Wie kann ich eine Mahnung anzeigen?
  • Wie markiere ich eine Mahnung als versendet?

Wozu dient die Tabelle?

Diese Tabelle zeigt automatisch Schüler, die bei der Rückgabe ihrer Lehrmittel in Verzug sind. Die Fristen für die Typen "erste Mahnung" und "zweite Mahnung" können hierbei in den Einstellungen festgelegt werden.

Wie kann ich eine Mahnung anzeigen?

Die Funktion "Mahnung anzeigen" erzeugt je nach Verzugstyp eine erste oder eine zweite druckbare Mahnung im PDF-Format. Diese richtet sich an den jeweiligen in der Tabelle ausgewählten Schüler oder (bei der 2. Mahnung) an dessen Eltern.

Wie markiere ich eine Mahnung als versendet?

Durch das Ausführen dieser Funktion wird der erfolgreiche Versand einer Mahnung bestätigt. Hierdurch wird der entsprechende Schüler aus der Tabelle entfernt. Wenn es sich hierbei um den Versand der ersten Mahnung gehandelt hat, so beginnt ab diesem Zeitpunkt die Frist bis zur Anzeige der zweiten Mahnung.
