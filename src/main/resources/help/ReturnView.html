Schüler durchsuchen

In diesem Feld kann man, indem man einen Schülernamen eingibt, nach einem speziellen Schüler suchen.

Daten auswählen

Die Liste „Daten auswählen“ stellt nach der erfolgreichen Suche eines Schülers im Feld „Schüler durchsuchen“ die Bücher bereit, die der Schüler zu erhalten hat.
Wenn keine Suche vor Nutzung dieser Liste stattgefunden hat, werden alle Klassen angezeigt. Durch Klick auf eine Klasse klappt eine Sub-Liste auf, die alle Schüler dieser Klasse anzeigt. Durch das Klicken auf einen Schüler klappt wiederrum eine Sub-Liste auf, welche alle auszugebenden Bücher für den Schüler enthält.

Ausgewählte Bücher zurückgegeben

Durch das anklicken dieses Buttons werden die in der Liste „Daten auswählen“ ausgewählte Bücher als zurückgegeben vermerkt.

Schülerliste für Auswahl drucken

Durch Klick auf diesen Button werden Bücherrückgabelisten für die/den gewählten Schüler (bzw. die gewählte Klasse/n) gedruckt.
