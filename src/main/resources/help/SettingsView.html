Benutzer

Mithilfe der Felder (Abbildung 1 - Punkt 1 und 2) und dem anschließenden anklicken des Speichern-Buttons (Abbildung 1 - Punkt 3), können Sie den Nutzernamen und die E-Mail-Adresse des Nutzers festlegen. Außerdem kann in diesem Tab das Kennwort für den Log-In geändert werden (Abbildung 1 - Punkt 4).

Abbildung 1: Nutzereinstellungen

Lehrmittelkategorien

In diesem Tab können Sie Lehrmittelkategorien erstellen (Abbildung 2 - Punkt 2), bearbeiten (Abbildung 2 - Punkt 4) und entfernen (Abbildung 2 - Punkt 3). Zum erstellen und bearbeiten von Lehrmitteln müssen Sie zuvor ein Lehrmittel aus der Tabelle (Abbildung 2 - Punkt 5) markieren.

Abbildung 2: Verwalten von Lehrmittelkategorien

Schuljahre

In diesem Tab können Sie Schuljahre erstellen (Abbildung 2 - Punkt 2), bearbeiten (Abbildung 2 - Punkt 4) und entfernen (Abbildung 2 - Punkt 3). Zum erstellen und bearbeiten von Lehrmitteln müssen Sie zuvor ein Lehrmittel aus der Tabelle (Abbildung 2 - Punkt 5) markieren.
Diese Definition der Schuljahre ist notwendig, damit diese Software später automatisiert Listen von Klassen und Schülern erstellen kann, die Lehrmittel benötigen oder Lehrmittel nicht länger benötigen.

Abbildung 3: Verwaltung von Schuljahren

Weitere Einstellungen

In diesem Tab können Sie durch markieren des Eintrags dun_firstDunningDeadline (siehe Abbildung 4 - Punkt 3) und darauf folgendes anklicken des Buttons Bearbeiten (siehe Abbildung 4 - Punkt 2) eine eigene erste Mahnfrist festlegen. Die Mahnfrist für die zweite Mahnung definieren Sie auf die gleiche Art und Weise, hier müssen Sie lediglich den Eintrag dun_secondDunningDeadline markieren.
